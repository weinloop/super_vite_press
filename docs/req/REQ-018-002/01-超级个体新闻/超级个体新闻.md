# 模块一：超级个体新闻

**REQ-ID**: REQ-018-002 | **优先级**: P0

---

## 模块概述

为用户提供「个体新闻」与「AI 新闻」两类内容的聚合展示，数据来源于每日定时爬取的外部站点，经去重、分类后在前端以列表/流形式展示，支持按分类筛选与基础检索（可选首版）。

**用户故事摘要**：内容创作者/独立开发者可快速获取个体创业、创作、AI 产品与技术相关的最新新闻；用户可区分两类内容、按分类筛选，并通过标题/摘要/来源/时间浏览，跳转原文。

### 信息结构

## 字段说明（输入、输出、必填、格式、长度、默认值）

### 新闻列表项（输出）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| id | 新闻唯一标识 | 是 | 字符串 | - |
| title | 标题 | 是 | 字符串，建议 ≤200 字 | - |
| summary | 摘要 | 否 | 字符串，列表可截断（如 120 字） | 空 |
| sourceName | 来源名称 | 是 | 字符串 | - |
| sourceUrl | 原文链接 | 是 | URL | - |
| publishTime | 发布时间 | 是 | ISO 8601 或时间戳 | - |
| category | 分类 | 是 | 枚举：individual / ai | - |

### 列表查询（输入）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| category | 分类 | 是 | individual \| ai | - |
| page | 页码 | 否 | 正整数 | 1 |
| pageSize | 每页条数 | 否 | 正整数，上限如 50 | 20 |
| keyword | 搜索关键词（P2） | 否 | 字符串 | - |

### 数据源配置（后台/输入）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| name | 来源名称 | 是 | 字符串 | - |
| urlRule | 采集 URL 或规则 | 是 | 字符串/URL | - |
| category | 归属分类 | 是 | individual \| ai | - |
| enabled | 是否启用 | 否 | 布尔 | true |

## 交互逻辑

- 进入新闻首页：默认展示「个体新闻」分类第一页数据。
- 切换 Tab：切换至「AI 新闻」或「个体新闻」时，刷新列表为对应分类第一页。
- 下拉刷新：重新请求当前分类第一页，替换当前列表。
- 分页/加载更多：请求下一页，追加到当前列表（或替换为分页控件翻页）。
- 点击列表项：进入详情页或直接在新窗口打开原文链接；若为详情页，则详情内提供「打开原文」按钮。
- 原文链接：新窗口打开，保留来源与合规说明展示。

## 按钮 / 操作行为

| 按钮/操作 | 位置 | 行为 |
|-----------|------|------|
| 个体新闻 / AI 新闻 Tab | 顶部 | 切换分类并刷新列表为第一页 |
| 列表项点击 | 列表 | 进入详情或打开原文（行为需统一） |
| 打开原文 / 查看原文 | 详情页 | 新窗口打开 sourceUrl |
| 加载更多 | 列表底部 | 请求下一页并追加到列表 |
| 下拉 | 列表区 | 触发下拉刷新，重载第一页 |

---

## 校验规则

- **列表查询**：category 必填且仅为 `individual` 或 `ai`；page、pageSize 为正整数，pageSize 不超过配置上限（如 50）。
- **数据源配置**：name、urlRule、category 必填；urlRule 为合法 URL 或可解析规则；enabled 为布尔。
- **爬虫**：仅爬取白名单内站点；遵守 robots.txt 与站点条款；优先使用官方 API。

## 异常处理与提示文案

| 场景 | 处理方式 | 提示文案（示例） |
|------|----------|------------------|
| 列表接口失败 | 展示空状态或重试入口 | 「加载失败，请下拉刷新重试」 |
| 无数据 | 展示空状态 | 「暂无该分类新闻」 |
| 原文链接失效 | 详情页仍展示，按钮可置灰或提示 | 「原文链接暂时不可用」 |
| 爬取/数据源异常 | 后台告警；前台可展示最近一次成功数据 | 不强制前台提示，可记录日志 |
| 网络超时 | 可重试 | 「网络超时，请重试」 |

## 分页、排序、筛选、搜索规则

| 类型 | 规则 |
|------|------|
| **分页** | 采用页码+每页条数（page + pageSize），默认 page=1、pageSize=20；支持「加载更多」时即请求 nextPage。 |
| **排序** | 列表按发布时间倒序（最新优先）。 |
| **筛选** | 仅分类筛选：个体新闻 / AI 新闻，通过 Tab 切换，对应 category 参数。 |
| **搜索** | 首版可选（P2）：在标题、摘要中关键词搜索，keyword 参数；支持与 category、分页组合。 |
