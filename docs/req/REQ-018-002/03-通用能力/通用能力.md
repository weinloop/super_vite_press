# 模块三：通用能力

**REQ-ID**: REQ-018-002 | **优先级**: P1

---

## 模块概述

包含个人设置、平台介绍、帮助与说明、以及合规相关页面，保证基础体验与合规。为 Web 与 APP 共用能力，设置入口、关于我们、帮助/FAQ、用户协议与隐私政策均需在首版就绪。

---

## 功能点清单

| ID | 功能点 | 优先级 | 说明 |
|----|--------|--------|------|
| C-1 | 设置入口 | P1 | Web 与 APP 均提供「设置」或「个人中心」入口，位置符合常见产品习惯 |
| C-2 | 通知设置 | P1 | 若支持消息推送，提供总开关及分类开关，用户可关闭/开启推送 |
| C-3 | 主题与显示 | P1 | 可选浅色/深色主题、字体大小等，首版可仅支持主题切换 |
| C-4 | 账号与安全 | P1 | 若支持登录，提供账号信息展示、修改与登出；登出后清除本地登录态 |
| C-5 | 平台介绍 | P1 | 「关于我们」页：产品定位、核心功能、联系方式（可选），内容可配置更新 |
| C-6 | 帮助/FAQ | P1 | 常见问题与使用说明，列表+展开或独立帮助页，至少覆盖：如何看新闻、如何用教程、如何反馈 |
| C-7 | 反馈入口 | P1 | 用户可提交意见或 Bug 反馈（表单或邮件链接），提交后有明显成功提示或邮件确认 |
| C-8 | 用户协议 | P1 | 注册/使用前可查看，在注册流程或设置中可随时查看 |
| C-9 | 隐私政策 | P1 | 说明数据收集、使用、存储与第三方共享，在注册或首次使用可查看并可从设置进入 |

## 界面原型 / 页面说明

### 页面清单

| 页面 | 说明 |
|------|------|
| 设置页 | 列表式菜单，含子项：通知、主题、账号、关于、帮助、隐私与协议 |
| 通知设置 | 总开关 + 分类开关（若有多类推送） |
| 主题/显示设置 | 主题切换（浅色/深色）、可选字体大小 |
| 账号与安全 | 账号信息展示、修改入口、登出按钮 |
| 关于我们 | 静态内容页：产品定位、核心功能、联系方式 |
| 帮助/FAQ | 列表+展开或独立帮助页 |
| 用户协议 | 静态内容页，可从设置或页脚进入 |
| 隐私政策 | 静态内容页，可从设置或页脚、注册流程进入 |
| 反馈页/弹层 | 表单或邮件链接入口 |

### 页面布局核心结构（用于生成布局示意图）

#### 设置页（入口页）

- **顶部导航区**：返回（APP）/页面标题「设置」（Web 可为面包屑）
- **中间内容区**：设置项列表（分组可选）：通知、主题、账号与安全、关于我们、帮助/FAQ、隐私政策、用户协议、反馈
- **底部操作区**：通常无固定操作栏（如有登出，放在「账号与安全」子页底部）

#### 典型子页（通知 / 主题 / 账号）

- **顶部导航区**：返回 + 子页标题
- **中间内容区**：表单/开关/选项列表
- **底部操作区**：需要“保存”时提供底部主按钮「保存」（也可自动保存，则不展示）

#### 静态内容页（关于/帮助/协议/隐私）

- **顶部导航区**：返回 + 标题
- **中间内容区**：正文内容（支持滚动）；帮助页为 FAQ 折叠列表
- **底部操作区**：通常无（可选“联系客服/反馈”入口）

#### 反馈页/弹层

- **顶部导航区**：返回/关闭 + 标题「反馈」
- **中间内容区**：类型选择（可选）+ 文本输入框 + 联系方式（可选）
- **底部操作区**：主按钮「提交」

## 字段说明（输入、输出、必填、格式、长度、默认值）

### 通知设置（输入/输出）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| enabled | 推送总开关 | 是 | 布尔 | true |
| categories | 分类开关（若有多类） | 否 | 对象，key 为分类标识，value 为布尔 | {} |

### 主题/显示设置（输入/输出）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| theme | 主题 | 否 | light \| dark \| system | system |
| fontSize | 字体大小（若支持） | 否 | 枚举（如 small/medium/large） | medium |

### 账号信息（输出，若支持登录）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| userId | 用户 ID | 是 | 字符串 | - |
| nickname | 昵称 | 否 | 字符串 | - |
| avatar | 头像 URL | 否 | URL | - |
| account | 账号（脱敏展示） | 否 | 字符串 | - |

### 反馈表单（输入）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| type | 类型 | 否 | 枚举：suggestion \| bug \| other | suggestion |
| content | 内容 | 是 | 字符串，建议长度上限 500～2000 | - |
| contact | 联系方式（可选） | 否 | 字符串，邮箱或手机 | - |

### 静态页（关于/帮助/协议/隐私）

- 以内容管理或静态配置输出，无结构化字段要求；需支持后台或配置更新文案。

## 交互逻辑

- 设置入口：从主导航或个人中心进入设置页，列表项点击进入对应子页或弹层。
- 通知/主题/账号：修改后即时生效（主题/字体）或保存后生效（通知开关）；登出后清除本地登录态并跳转至未登录态首页或登录页。
- 关于我们/帮助/协议/隐私：点击即进入静态内容页，支持从设置或页脚多处进入；协议与隐私在注册流程中需可查看。
- 反馈：填写表单提交或打开邮件客户端；提交成功后展示成功提示或邮件确认说明。

---

## 按钮 / 操作行为

| 按钮/操作 | 位置 | 行为 |
|-----------|------|------|
| 设置 / 个人中心 | 主导航/头部 | 进入设置页 |
| 通知设置 / 主题 / 账号 / 关于 / 帮助 / 隐私与协议 | 设置列表 | 进入对应子页或弹层 |
| 推送总开关、分类开关 | 通知设置页 | 切换后保存并生效 |
| 主题选项 | 主题设置页 | 切换后即时生效 |
| 登出 | 账号与安全页 | 清除登录态并跳转 |
| 用户协议 / 隐私政策 | 设置、页脚、注册流程 | 打开静态页 |
| 提交反馈 | 反馈表单 | 提交后展示成功提示或邮件确认 |
| 帮助项 | 帮助页 | 展开/收起或跳转说明 |

---

## 校验规则

- **通知/主题**：enabled 为布尔；theme 为约定枚举；fontSize 若存在则为约定枚举。
- **反馈**：content 必填，长度不超过配置上限（如 2000 字）；contact 若填则符合邮箱或手机格式。
- **账号**：登出时清除本地 Token/登录态，并可选调用后端登出接口。

---

## 异常处理与提示文案

| 场景 | 处理方式 | 提示文案（示例） |
|------|----------|------------------|
| 设置保存失败 | 保留当前界面，提示失败 | 「保存失败，请重试」 |
| 登出失败（网络等） | 仍清除本地登录态，可提示 | 「已退出登录」或「网络异常，已本地退出」 |
| 反馈提交失败 | 保留表单内容，提示重试 | 「提交失败，请检查网络后重试」 |
| 反馈提交成功 | 清空表单并展示成功态 | 「感谢反馈，我们会尽快处理」 |
| 协议/隐私页加载失败 | 展示错误态或重试 | 「内容加载失败，请稍后重试」 |

---

## 分页、排序、筛选、搜索规则

| 类型 | 规则 |
|------|------|
| **分页** | 设置、关于、帮助、协议、隐私均为单页或静态内容，无需分页。FAQ 若条目较多可采用列表+展开或分页，每页条数可配置。 |
| **排序** | 设置菜单项、帮助/FAQ 条目可按配置顺序展示。 |
| **筛选** | 不适用。 |
| **搜索** | 帮助/FAQ 可选支持关键词搜索；首版可不做。 |
