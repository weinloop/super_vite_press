# 模块二：如何成为超级个体（教程）

**REQ-ID**: REQ-018-002 | **优先级**: P1

---

## 模块概述

提供结构化教程/指南，帮助用户理解并实践「成为超级个体」的路径，包括阶段划分、主题分类、精选内容聚合，可与新闻/案例联动（如「本周案例」）。内容支持自产、精选转载（带来源）与外链聚合，外链需标明来源。

**用户故事摘要**：新手可看到清晰的阶段划分与下一步指引；用户可按主题浏览教程；教程含标题、摘要、正文或外链，并可与新闻/案例关联。

---

## 功能点清单

| ID | 功能点 | 优先级 | 说明 |
|----|--------|--------|------|
| T-1 | 阶段结构展示与导航 | P1 | 多阶段（如入门→进阶→变现），每阶段下挂载多篇教程或链接 |
| T-2 | 主题分类与筛选 | P1 | 多主题（工具、AI 协作、内容创作、一人公司、政策与合规等），支持按主题筛选/导航 |

| T-4 | 教程详情 | P1 | 标题、摘要、正文或外链、可选配图、可选「关联新闻/案例」 |
| T-5 | 内容来源标识 | P1 | 自产/转载/外链均标明来源，符合版权约定 |
| T-6 | 与新闻联动（可选） | P1 | 教程详情或专区展示「相关新闻」「本周案例」，数据来自新闻模块，可配置展示规则 |

### 信息结构

- **教程首页**
  - 阶段 Tab 或步骤条：入门 → 进阶 → 变现（可配置）
  - 主题筛选项：工具、AI 协作、内容创作、一人公司、政策与合规等
  - 教程卡片列表：标题、摘要、主题标签
- **教程详情**
  - 标题、摘要
  - 正文内容或「打开外链」按钮
  - 可选配图
  - 可选「相关新闻」「本周案例」区块（无则隐藏）

### 界面要点

- 阶段与教程关联关系可配置；主题列表可配置；一篇教程可属多主题
- 外链在新窗口打开；每条内容有明确来源标识

## 字段说明（输入、输出、必填、格式、长度、默认值）

### 阶段（输出）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| id | 阶段 ID | 是 | 字符串 | - |
| name | 阶段名称 | 是 | 字符串 | - |
| order | 排序序号 | 否 | 正整数 | 0 |

### 主题（输出）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| id | 主题 ID | 是 | 字符串 | - |
| name | 主题名称 | 是 | 字符串 | - |
| order | 排序序号 | 否 | 正整数 | 0 |

### 教程列表项（输出）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| id | 教程 ID | 是 | 字符串 | - |
| title | 标题 | 是 | 字符串 | - |
| summary | 简短摘要 | 是 | 字符串 | - |
| stageId | 所属阶段 ID | 否 | 字符串 | - |
| topicIds | 主题 ID 列表 | 否 | 字符串数组 | [] |
| coverImage | 配图 URL | 否 | URL | - |
| sourceType | 来源类型 | 是 | 枚举：self / repost / external | - |
| sourceLabel | 来源标识文案 | 否 | 字符串 | - |
| externalUrl | 外链地址（外链类型时） | 否 | URL | - |

### 教程详情（输出）

在列表项基础上增加：

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| content | 正文（富文本或 Markdown） | 否 | 字符串 | - |
| relatedNews | 相关新闻/本周案例列表 | 否 | 数组，结构同新闻列表项 | [] |

### 列表/筛选（输入）

| 字段 | 说明 | 必填 | 格式/长度 | 默认值 |
|------|------|------|-----------|--------|
| stageId | 阶段 ID | 否 | 字符串 | - |
| topicId | 主题 ID | 否 | 字符串 | - |
| page | 页码 | 否 | 正整数 | 1 |
| pageSize | 每页条数 | 否 | 正整数 | 10 |

## 交互逻辑

- 进入教程首页：展示默认阶段（如「入门」）及全部或默认主题下的教程列表。
- 点击阶段 Tab/步骤：切换阶段，刷新教程列表为当前阶段下的教程。
- 选择主题筛选：筛选当前阶段（或全部）下属于该主题的教程，刷新列表。
- 点击教程卡片：进入教程详情页。
- 详情页为外链类型：展示摘要与「打开外链」按钮，点击在新窗口打开 externalUrl。
- 详情页为自产/转载：展示正文 content；若有「相关新闻」「本周案例」配置则展示对应区块。
- 相关新闻/案例：可点击跳转新闻详情或原文。

## 按钮 / 操作行为

| 按钮/操作 | 位置 | 行为 |
|-----------|------|------|
| 阶段 Tab / 步骤条 | 教程首页顶部 | 切换阶段并刷新教程列表 |
| 主题筛选项 | 教程首页 | 按主题筛选并刷新列表 |
| 教程卡片 | 列表 | 进入教程详情页 |
| 打开外链 / 阅读原文 | 教程详情 | 新窗口打开 externalUrl |
| 相关新闻/案例条目 | 详情页区块 | 跳转新闻详情或原文 |

## 校验规则

- **列表查询**：stageId、topicId 若传则需为有效 ID；page、pageSize 为正整数，pageSize 不超过上限（如 30）。
- **教程内容**：title、summary 必填；sourceType 为 self | repost | external；为 external 时 externalUrl 必填且为合法 URL；content 与 externalUrl 二选一或按 sourceType 约束。
- **阶段/主题**：name 必填；order 为非负整数。

---

## 异常处理与提示文案

| 场景 | 处理方式 | 提示文案（示例） |
|------|----------|------------------|
| 教程列表接口失败 | 展示空状态或重试 | 「加载失败，请稍后重试」 |
| 无教程数据 | 展示空状态 | 「该分类下暂无教程」 |
| 教程详情不存在 | 404 或跳转列表 | 「教程不存在或已下架」 |
| 外链无法打开 | 新窗口打开失败时 | 「链接暂时无法打开，请稍后重试」 |
| 相关新闻/案例无数据 | 隐藏该区块 | 不展示区块 |

---

## 分页、排序、筛选、搜索规则

| 类型 | 规则 |
|------|------|
| **分页** | 采用 page + pageSize，默认 page=1、pageSize=10；支持加载更多或分页控件。 |
| **排序** | 教程列表按配置顺序或发布时间/排序字段倒序，具体可配置。 |
| **筛选** | 按阶段（stageId）、主题（topicId）筛选；可组合使用。 |
| **搜索** | 首版可不做教程内全文搜索；若做则支持标题、摘要关键词搜索。 |
