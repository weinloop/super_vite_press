# UX 摘要（REQ-018-002｜超级个体新闻）

## 主要功能（P0）

- **双分类聚合**：提供「个体新闻」与「AI 新闻」两类内容的聚合展示。
- **高效浏览**：以列表/流形式呈现，列表项包含标题、摘要、来源、时间。
- **低成本深读**：点击进入详情页查看摘要/合规说明，并一键跳转原文。
- **持续更新**：支持刷新与加载更多，快速获取最新内容。

## 目标用户与场景

- **目标用户**：内容创作者、独立开发者、个体创业者、对 AI 产品/技术动态敏感的从业者。
- **典型场景**：
  - 通勤/碎片时间快速扫读当天热点。
  - 开工前 5 分钟快速抓取行业信号。
  - 看到标题后进入详情确认信息，再打开原文深读。

## 核心交互逻辑（主流程）

浏览（默认进入个体新闻） → 切换分类（个体/AI） → 滚动列表扫读 → 点击进入详情 → 打开原文（新窗口/浏览器）  

关键分支：

- **刷新**：回到当前分类第一页，替换当前列表内容（原型以“刷新按钮/动效”模拟）。
- **加载更多**：请求下一页，追加到列表末尾。
- **空状态**：该分类暂无新闻。
- **错误/超时**：提示可重试（原型提供可切换的模拟状态）。

## 成功指标（体验目标）

- **少步骤完成**：从进入页面到打开原文 ≤ 2 次点击（列表项 → 详情 → 打开原文，或列表项直达原文）。
- **可扫读**：列表项一屏可读 3–6 条（取决于摘要长度），标题与时间清晰。
- **一致性**：两类内容的结构与交互一致，降低学习成本。
