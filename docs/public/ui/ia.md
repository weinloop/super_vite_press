# 信息架构（REQ-018-002｜超级个体新闻）

## 页面清单

- **入口页**：`ui/index.html`（原型导航）
- **新闻首页（列表）**
  - iOS：`ui/mobile/ios.html`
  - Web：`ui/web/desktop.html`
- **新闻详情页**
  - iOS：`ui/mobile/ios-detail.html`
  - Web：`ui/web/detail.html`

## 页面层级与导航关系

- 入口页
  - → iOS 列表页
    - → iOS 详情页
      - → 打开原文（外链新窗口）
  - → Web 列表页
    - → Web 详情页
      - → 打开原文（外链新窗口）

## 关键界面说明

### 新闻首页（列表）

- **目的**：快速浏览最新新闻，按分类（个体/AI）切换并持续加载。
- **核心内容**：
  - 顶部：标题/（可选）搜索入口、刷新入口
  - 中间：分类 Tab「个体新闻 | AI 新闻」
  - 列表：卡片项（标题、摘要截断、来源、发布时间）
  - 底部：加载更多（或分页）
- **主要操作**：切换分类、刷新、加载更多、点列表项进详情。

### 新闻详情页

- **目的**：在合规前提下展示摘要与来源信息，并引导用户打开原文深读。
- **核心内容**：
  - 标题、来源、发布时间
  - 摘要/正文片段（Mock）
  - 合规说明（来源与版权提示）
  - 主按钮：打开原文（外链）
- **主要操作**：返回列表、打开原文。
